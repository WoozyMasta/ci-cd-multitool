# CI/CD multitool

[Docker image](https://hub.docker.com/r/woozymasta/ci-cd-multitool)
with useful tools for CI/CD pipelines

Based on Alpine

```bash
docker pull woozymasta/ci-cd-multitool:latest
```

Include:

* bash
* openssh-client
* git
* tar
* xz
* gzip
* bzip2
* curl
* coreutils
* rsync
* openssl
* kubectl
* jq
* yq
